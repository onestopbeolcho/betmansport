## 📌 1. 보상 토큰의 세 가지 구현 방식

### Option A: 기존 코인 보상 (BTC/ETH/USDT)
> 예측 적중 시 소량의 비트코인/이더리움/스테이블코인 지급

| 항목 | 평가 |
|------|------|
| 구현 난이도 | ★★★☆☆ |
| 유저 매력도 | ★★★★★ (실제 가치) |
| 법적 리스크 | 🔴 **매우 높음** |
| 비용 부담 | 🔴 **높음** (직접 코인 구매 필요) |

**법적 문제:**
- 한국: 가상자산사업자(VASP) 신고 의무 → 실명확인 입출금계정, 보안 인증 필요
- 미국: 돈세탁방지법(AML) + 자금이체업(MSB) 등록 의무
- EU: MiCA CASP 라이선스 필요
- **결론: ❌ 스타트업에는 사실상 불가능한 진입 장벽**

### Option B: 자체 유틸리티 토큰 발행 (SNX Token)
> Scorenix 플랫폼 전용 토큰 발행, 플랫폼 내에서만 사용

| 항목 | 평가 |
|------|------|
| 구현 난이도 | ★★★★☆ |
| 유저 매력도 | ★★★★☆ (토큰 가치 상승 기대) |
| 법적 리스크 | 🟡 **중간~높음** (설계에 따라 다름) |
| 비용 부담 | 🟢 **낮음** (스마트 컨트랙트 배포 비용만) |

**핵심 법적 판단: Utility Token vs Security Token**

```
✅ Utility Token으로 인정받으려면:
├─ 투자 수익을 약속하지 않을 것
├─ 플랫폼 내 서비스 접근 용도로만 사용
├─ 발행 시점에 플랫폼이 이미 운영 중일 것
├─ 외부 거래소 상장 없이 플랫폼 내에서만 유통
└─ 토큰 가격 상승을 마케팅하지 않을 것

❌ Security Token으로 분류되면:
├─ SEC 등록 의무 (미국)
├─ MiCA 전면 규제 (EU)
├─ 한국 가상자산법 전면 적용
└─ 사실상 증권사 수준의 컴플라이언스 필요
```

### 🇺🇸 미국

#### SEC (증권거래위원회)
| 기준 | 내용 |
|------|------|
| Howey Test | ① 금전 투자 ② 공동 사업 ③ 타인 노력에 의한 수익 기대 → 3가지 모두 충족 시 '증권' |
| 2026 Innovation Exemption | 컴플라이언스 기준 충족 시 간소화된 등록 가능 |
| Meme Coin 가이드라인 | "오락/문화 목적 구매 토큰은 일반적으로 증권 아님" |

#### CFTC (상품선물거래위원회)
- 스포츠 예측 시장 ≈ 이벤트 계약 ≈ 선물거래로 분류
- Kalshi, Polymarket 등 규제 하 운영 중
- **스포츠 예측 + 코인 보상 = SEC + CFTC + 주(州) 도박법 3중 규제 가능**

#### 주(州)별 분쟁
- 네바다, 매사추세츠: 예측 시장 = 도박으로 분류 → 소송 진행 중
- **결론: 미국 시장은 법적 불확실성 매우 높음**

## 🎯 4. Scorenix 최적 전략 제안

### 단계별 접근법 (Progressive Decentralization)

```
Phase 1 (현재~6개월)          Phase 2 (6~12개월)          Phase 3 (12개월+)
─────────────────          ─────────────────          ─────────────────
SP 포인트 시스템             블록체인 투명성 추가         토큰 전환 검토
(중앙화, 법적 안전)           (포인트 기록 온체인)          (규제 환경 확인 후)

✅ 법적 리스크 없음           ✅ 기술적 신뢰도 ↑          ⚠️ 법적 검토 필수
✅ 즉시 구현 가능             ✅ 마케팅 차별화             ⚠️ VASP 등록 시
✅ 빠른 시장 검증             ✅ 포인트 ≠ 코인 (합법)      ⚠️ AML/KYC 강화
```

### Phase 1: SP 포인트 (즉시 구현)
- 기존 계획대로 Scorenix Points (SP) 운영
- 플랫폼 내부 보상(구독권, 배지, 기능 해금)
- **법적 리스크: 거의 없음** (내부 포인트 = 규제 대상 아님)
- 외부 현금화 경로 차단 (법적 핵심!)

### Phase 2: 블록체인 기록 (6개월 후)
```
┌─────────────────────────────────────────┐
│  "Your predictions are recorded on-chain │
│   for full transparency and fairness"   │
│                                         │
│  📊 내 예측 기록: 0x3f2a...hash         │
│  🏆 리더보드 검증: Block #4,521,837     │
│  ✅ 위변조 불가능한 공정 시스템           │
└─────────────────────────────────────────┘
```
- SP 적립/사용 기록을 블록체인에 해시로 기록
- "코인"은 아니지만 블록체인 투명성을 마케팅으로 활용
- Base, Polygon 등 L2 체인 활용 (저렴한 가스비)
- **법적 리스크: 낮음** (기록 목적 = 가상자산 아님)

### Phase 3: 토큰 전환 (12개월+ / 조건부)

> [!WARNING]
> 다음 조건이 모두 충족될 때만 진행:
> 1. 한국 디지털자산 기본법(2단계) 시행 + 규제 명확화
> 2. 법률 자문 확보 (가상자산 전문 로펌)
> 3. 충분한 유저 베이스 (MAU 10,000+)
> 4. VASP 등록 가능한 자본/인프라 확보
